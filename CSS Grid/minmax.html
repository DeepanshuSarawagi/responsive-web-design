How to implement minmax function

We can use the minmax function to limit the size of items in the grid container. grid-templatr-rows and grid-template-cloumns accept the property of minmax(90px, 200px) which means it will create a column with min width as 90 px and maximum width of 200px. Similarly if used in rows, it will create a row with minimum height of 90px and maximum height of 200 px.

consider that we need 100 rows in a grid area and 50 cloumns with same size. It would be practically impossible to enter 100 row values of different or same size. Hence we can use the repeat function to create such. grid-template-rows: repeat(100, 50px); will create 100 rows with 50px tall. grid-template-columns: repeat(50, 100px); will create 50 columns with 100px in width. We can also enter multiple values in following way grid-template-rows: repeat(2, 100px 50px) 75px; This will create rows with follwing values 100px 50px 100px 50px 75px.

In CSS Grid, content of each item which is located in a box is referred to as cell. We can justify the content of the cell by adding the justify-self property. default accepted value is stretch and it spans the entire content in the entire cell. justift-self aligns the item horizantally.

justify-self accepts below value:

  • start: Aligns the content to the left of the cell.
  • end: Aligns the content to the right of the cell.
  • center: Aligns the content to the center of the cell.

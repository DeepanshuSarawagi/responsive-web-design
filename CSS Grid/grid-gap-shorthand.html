We can use the shorthand property for the grid-gap which accepts values in px , fr, and %. Instead of specifying the grid-row-gap and grid-column-gap we can use the grid-gap: 10px which will automatically add gap of 10px in all rows and columns. You can also specify the grid-gap: 10px 20px; which will add 10px gap in rows and 20px gap in columns.

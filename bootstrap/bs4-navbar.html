Navigation Bar in Bootstrap 4

Basic Navigation Bar

Navigation bar in bootstrap 4 can collapse or extend based on the screen size. A standard navigation bar is created with class .navbar followed by a responsive collapsin class .navbar-expand-xl|lg|md|sm (stacks the navbar vertically on extra large, large, medium or small screens).

To add the links inside navbar add the class .navbar-nav in the <ul> element and add the class .nav-item in the <li> element and class .nav-link in <a> element.

Collapsing the Navigation Bar

Collapsing the navigation bar can be done by using the class class="navbar-toggler" and data-toggle="collapse" and data-target="#thetarget" in the button element. The wrap the navbar contents (links, etc) in the <div> element by adding the class class="collapse navbar-collapse" and id="thetarget"

Forms and Buttons in Navigation Bar

To create Forms in Navigation Bar we need to add a <form> element wrapped inside <nav> bar. The form element should contain class .form-inline and with an action attribute. Then create an <input> element with class .form-control and mr-sm-2 and type="text" and give a placeholder. Also wrap a button element inside form.

We can also use other input classes such .input-group-prepend or .input-group-append to create an icon next to the input field.

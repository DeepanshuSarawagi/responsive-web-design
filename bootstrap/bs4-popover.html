Bootstrap 4 Popover

The Popover component is similar to Tooltip; when a user clicks on the element a small popup box appears. The difference is that a popover can contain a lot more content.

How to create Popover?

We can create popover by using the attribute data-toggle="popover" and title for the popover header and data-content="some content" attribute for adding content inside the Popover.

Positioning Popovers

By default, popovers will position on the right side of the element. If you want to customize the position of popover then use the data-placement="right/left/top/bottom" attribute in the element.

Closing Popovers

By default Popover is closed when the element is clicked again. But we can use data-trigger="focus" attribute when we click anywhere in the document.

Click anywhere to close me!

Tip: If you want the popover to appear when the user hovers over the element then use data-trigger="hover" attribute in the element.

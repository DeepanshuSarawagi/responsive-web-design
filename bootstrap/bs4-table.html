Contextual Classes Tables:

We can create tables with contextual colors. Either whole <table> or table rows <tr> or table cells <td> can be colored using the contextual classes. Following classes can be used to color the table and its elements.

  • .table-primary
  • .table-success
  • .table-warning
  • .table-danger
  • .table-info
  • .table-active
  • .table-secondary
  • .table-light
  • table-dark

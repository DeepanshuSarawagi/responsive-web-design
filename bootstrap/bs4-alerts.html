Alerts

Alerts are created with class .alert followed by contextual color classes.

Success! This alert box could indicate a success/positive action.
Warning! This alert box could indicate a warning that might need attention.
Danger! This alert box could indicate a dangerous/potentially negative action.
Info! This alert box could indicate a neutral/informative action.
Secondary! This alert box could indicates a slightly less important message.

Alert Links

We can create alert links by using the class .alert-link inside the alert box.

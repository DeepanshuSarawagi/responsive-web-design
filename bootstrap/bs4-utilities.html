Rounded elements

We can have the elements rounded by using the .rounded class.

Float and Clearfix

Float an element to the right with class .float-right and left with class .float-right. We can have the element/item with clear floats using the class .clearfix.

Hello! I`ll be on right Hello! I`ll be on left
Hello! I`m clear floats

Responsive Floats

We can float an element responsively depending on the screen width by adding the class float-*-right|top whereas * denotes sm|md|lg|xl.

Hello! I`ll be on the right on small screens.

Hello! I`ll be on the right on medium screens.

Hello! I`ll be on the right on large screens.

Hello! I`ll be on the right on extra large screens.

We can center align the elements by using the class .mx-auto.

Width

Set the width of the element by using w-* classes. For e.g., w-25 will create a width of 25%, w-50, w-75, w-100 will create width of element 50%, 75% and 100% respectively.

Bootstrap 4 Flexbox

The biggest difference between bootstrap 3 and bootstrap 4 is that the latter uses the flexbox instead of float to handle the layout.

The flexible layout module makes it easier to design flexible responsive layout structure without using float or positioning. To create flex container and to transform the children to flex items, use the class d-flex .

Responsive Flex Classes

All flex classes comes with additional responsive classes, which makes it easy to set a specific class on a specific screen size. * symbol can be replaced with sm|md|lg|xl which represents small, medium, large and extra large screens.

Class Description
Flex Container  
.d-*-flex Creates a flexbox container for different screen sizes.
.d-*-inline-flex Creates a inline flex container for different screens.
Flex Direction  
.flex-*-row Displays flex items horizontally on different screens
.flex-*-row-reverse Displays flex items horizontally and right aligned on different screens.
.flex-*-column Displays flex items vertically on different screens
.flex-*-column-reverse Displays flex items vertically and right aligned on different screens.
Justified Content  
.justify-content-*-start Displays flex items from the start (left aligned) on different screen sizes.
.justify-content-*-end Displays flex items at the end (right aligned) on different screen sizes.
.justify-content-*-center Displays flex items at the center of the flex Container on different screen sizes.
.justify-content-*-between Displays flex items in between of the flex Container on different screen sizes.
.justify-content-*-around Displays flex items around the flex Container on different screen sizes.
Fill/Equal Width  
.flex-*-fill Force flex items into equal widths on different screens.
Flex Grow  
.flex-*-grow-0 Don`t make the items grow on different screens.
.flex-*-grow-1 Makes the flex items grow on different screens.
Flex Shrink  
.flex-*-shrink-0 Don`t make the flex items shrink on different screen sizes.
.flex-*-shrink-1 Make the flex items shrink on different screens.
Order  
.order-*-0-12 Change the order of flex items on small screens.
Flex Wrap  
.flex-*-nowrap Dont wrap items on different screens.
.flex-*-wrap Wrap flex items on different screens.
.flex-*-wrap-reverse Reverse the wrapping of flex items on different screens.
Align Content  
.align-content-*-start Align the gathered items from the start of different screens.
.align-content-*-end Align the gathered items from the end of different screens.
.align-content-*-center Align the gathered items from the center of different screens.
.align-content-*-around Align the gathered items around the container on different screens.
.align-content-*-stretch Stretch the gathered items on different screens.
Align Items  
.align-items-*-start Align single rows of items from the start on the different screens.
.align-items-*-end Align single rows of items from the end on the different screens.
.align-items-*-center Align single rows of items from the center on the different screens.
.align-items-*-baseline Align single rows of items from the baseline on the different screens.
.align-items-*-stretch Stretch single rows of items on the different screens.
Align Self  
.align-self-*-start Align a flex item from the start on different screens.
.align-self-*-end Align a flex item from the end on different screens.
.align-self-*-center Align a flex item from the center on different screens.
.align-self-*-baseline Align a flex item from baseline on different screens.
.align-self-*-stretch Stretch a flex item on different screens.

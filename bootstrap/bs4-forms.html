Bootstrap 4 Froms

Bootstrap 4`s default settings

Bootstrap 4 Form controls automatically receive some global styling with Bootstrap: All textual <input>, <textarea> and <select> elements with class .form-control have a width of 100%.

Bootstrap 4 Form Layouts

Bootstrap 4 comes with two layouts:

  • Stacked (full-width) form
  • Inline Form

Bootstrap Inline Form

By adding the class .form-inline in the <form> element will create the all the elements inline and left aligned. Note: This only applies for the viewports which are atleast 576px wide. For a viewport smaller than 576px, it will stack horizontally.

Inline Form with Utilities

The inline form which we created above looks congested and will look much better by utilising the bootstrap space utilities. By adding the class .mr-sm-2 adds right margin to each input on all the devices.

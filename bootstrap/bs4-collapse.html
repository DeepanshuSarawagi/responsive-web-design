Basic Collapsible

Collapsibles are useful when you want to show or hide large amount of data.

We can create collapse by using the pramater data-toggle="collapse" and data-target="#demo" in <button> element. Then create a <div> element with class .collapse and id #demo. This div element should have the paragraph which needs to be hidden or shown. Please refer to the below example.

How does the above example work?

We use the class .collapse in the <div> element to show/hide the content based on the click of a button. The data-toggle="collapse" can be used in <button> or <a> tag to control the collapsible content. Then add the data-target="#id" to connect the button with the collapsible content. If we want to use the anchor tag instead of instead of button element then add the id in href attribute of anchor tag.

Accordion

Accordions are collpasible items grouped together. We need to use the data-parent="accordion" attribute in the <div> element to make sure that all collapsible elements under the specified parent will be closed when one of the collapsible item is shown.

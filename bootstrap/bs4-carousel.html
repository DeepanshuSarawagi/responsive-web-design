Example explained

Below is the explanation for each class used in creating carousel and its use.

Class Name Description
.carousel Creates a carousel.
.carousel-indicators Adds indicators for the carousel. These are the little dots at the bottom of the slide which specifies how many slides are there in the carousel and which slide currently user is viewing.
.carousel-inner Adds slide to the carousel.
.carousel-item specifies the content of each slide.
.carousel-control-prev Adds a left button to the carousel which allows user to go back between the slides.
.carousel-control-next Adds a right button to the carousel which allows the user to go forward between the slides.
.carousel-control-prev-icon Used together with .carousel-control-prev to create the previous button.
.carousel-control-next-icon Used together with .carousel-control-next to create the forward button.
.slide Adds a CSS transition and animation effect when sliding from one item to the next.

Add captions to the side

We can add captions to each slide by using the class .carousel-caption in each .carousel-item element.

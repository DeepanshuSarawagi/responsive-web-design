Bootstrap 4 Custom Forms

Bootstrap 4 comes with custom forms which are meant to replace browser default form elements.

Custom Checkbox

To create a custom checkbox wrap a container element like <div> with the class .custom-control and .custom-checkbox. Then in the <input> element, use the class .custom-control-input. Note: If you are using a label associated with the checkbox the id attribute in input element should match with for attribute in the <label> element.

Custom Switch

To create a custom switch add the class .custom-control .custom-switch in the <div> element.

Custom Radio Buttons

To create custom radio buttons, add following classes like .custom-control and .custom-radio in the wrapper container like <div> . Then use the class .custom-control-input in the input element.
Note: If we are using the labels then the id attribute in the input element should match with the for attribute in the label element.

Buttons in Bootstrap 4

We can create buttons by using the class .btn followed by contextual color classes

Button Outline

Bootstrap 4 provides 8 outline/bordered buttons

Button Groups

Now, we are going to learn how to group buttons together. We need to use a class .btn-group in the <div> element.

Button sizes

We can also create different sizes of buttons by using the class .btn-group-lg or .btn-group-sm

Vertical Buttons

We can also create vertical buttons by using the class .btn-group-vertical

Dropdown menus in button

We can create dropdown menus in button by using the class .dropdown-toggle and data-toggle="dropdown" We also need to create a separate div element with class dropdown-menu and insert anchor tags with class dropdown-item

Split Button Dropdowns

We can create split button dropdowns by using an additional class .dropdown-toggle-split

Bootstrap 4 Tooltip

Tooltip is a small popup box which appears when a user hovers over the element.


How to create a Tooltip?

To create a tooltip add a data-target="tooltip" in the element and then add the title attribute to specify the text inside Tooltip.

Positioning Tooltip

We can position the tooltip by using the attribute data-placement="top/botton/right/left" in the element.

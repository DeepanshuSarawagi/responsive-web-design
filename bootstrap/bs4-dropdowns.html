Basic dropwdown in Bootstrap4

We can create basic basic dropdown by using .dropdown class in <div> element. To open the dropdown menu we need to use the class .dropdown-toggle and data-toggle="dropwdown" in <button> or <a> element.

Dropdown Divider

We can also create dropdown dividers by using the class .dropdown-divider to separate the links inside the dropdown menu with thin horizontal border.

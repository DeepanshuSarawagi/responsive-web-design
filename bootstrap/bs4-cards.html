Titles, Texts and Links

We can add titles in the card content by using the class .card-title inside .card-body. And we can add texts in cards by using the class .card-text in <p> element. We can add links links in the card by using the .card-link class inside <a> element.

Card Columns

We can create card columns by using the class card-columns which creates a masonry-like grid of cards (like pinterest). The layout will automatically adjust as you insert more cards.

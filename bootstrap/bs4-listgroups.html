List Group with Linked Items

To create a list group with Linked items, we need to use <div> element instead of <ul> element and <a> element instead of <li> element. Additionally we need to use the class .list-group.item-action if you want a grey background color on hover.

Apple Samsung Sony

Disabled Item

The .disabled class adds a lighter text color to the disabled item. And when used on links, it will remove the hover effect

Apple Samsung Sony

Inputs in Bootstrap 4

Supported Bootstrap 4 inputs

Bootstrap 4 supports below types of inputs

  • textarea
  • checkbox
  • radio
  • select
  • input

Bootstrap 4 Inputs

Bootstrap 4 supports supports all the html5 inputs: text, password, datetime, datetime-local date, month, time, week, number, email, url, search, tel and color.
Note: Input will not be fully styled of their type is not properly declared. The following example contains two input types: one for text and other is password. We will use the class .form-control for proper styling and padding.

Bootstrap 4 textarea

Textarea is a kind of input where you want user to user to provide feedback.

Bootstrap 4 checkboxes

Checkbox is a type of input when user to allowed to select multiple choices. We need to wrap a <label> element with class .form-check-label and <input> element with class .form-check-input inside <div> element with class .form-check.

Inline Checkboxes

Bootstrap Radio Buttons

Radio buttons are used if you want the user to limit to just one selection from the given list of choices.

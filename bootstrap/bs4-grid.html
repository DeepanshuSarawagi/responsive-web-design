Grid Classes

The Bootstrap 4 grid system has 5 classes:

  • .col- for extra small devices.
  • .col-sm- for small devices.
  • .col-md- for medium devices.
  • .col-lg- for large devices.
  • .col-xl- for extra large devices.

The classes above can be used to create more dynamic and flexible layouts.

Auto Layout Columns

In Bootstrap 4, there is an easy way to create equal width of columns. Instead of creating columns by specifying the number in the class .col-size-* just specify the .col-size and bootstrap will automatically distribute the width equally based on the number of columns created.

    Toggleable/Dynamic Tabs

    To make the navigation tabs toggleable, we need to add the attribute data-toggle="tab" in the each link. The create a class .tab-pane and wrap it inside <div> element with unique ID. Create multiple tab panes inside <a> element with class .tab-content. You can add the class active or fade in tab panes.

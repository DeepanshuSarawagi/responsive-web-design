Introduction:

HTML and CSS is a font-end web development language widely used to create websites and static pages. Latest browsers support HTML5 and CSS3. HTML, known as Hyper Text Markup Language is the standard markup language for documents designed to be displayed in a web browser. Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML Elements:

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page. HTML provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets. Tags such as <img/> and <input /> directly introduce content into the page. Other tags such as <p> surround and provide information about document text and may include other tags as sub-elements. Browsers do not display the HTML tags, but use them to interpret the content of the page.

There are several HTML elements. Most common ones used are:

  • The <!DOCTYPE html> declaration defines this document to be in HTML5.
  • The <html> element is the root element of an HTML page
  • The lang attribute defines the language of the document
  • The <meta> element contains meta information about the document
  • The charset attribure defines the character set used in the document
  • The <title> element specifies a title for the document
  • The <body> element contains the visible page content
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph

Anchor Tag:

The <a> tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

By default, links will appear as follows in all browsers:

  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red
You can use the target="_blank" inside the <a> tag to open the desitnation link in new tab. <a href="https://www.google.com" target="_blank">Google</a> You can test by clicking here.

Lists:

You can create two types of lists in HTML, Unordered lists and Ordered lists.

Syntax to create Unordered list is <ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul> Output of the above code/syntax would be
  • Item 1
  • Item 2
  • Item 3
Syntax to create Ordered list is <ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol> Output of the above code/syntax would be
  1. Item 1
  2. Item 2
  3. Item 3

What is CSS?

CSS stands for Cascading Style Sheets. It describes how HTML page should ba displayed on the browser. CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

HTML was never introduced to contain tags for formatting a web page. HTML was created to describe the content of the page. When tags like <font>, and color attributes were added to the HTML 3.2 specification, it started a nightmare for web developers. Development of large websites, where fonts and color information were added to every single page, became a long and expensive process. To solve this problem, the World Wide Web Consortium (W3C) created CSS.

CSS Syntax:

A CSS rule-set consists of a selector and a declaration block:

h1 {color: blue;
font-size: 12px;
} In the above code, h1 is the CSS selector, and the declaration in the curly braces are CSS property and its corresponding property.
In this example, all the <p> elements will be center-aligned with a red text color. p { color: red;
text-align: center;
} Always remember that the CSS declarations opens and closes with a curly braces and the property and the corresonding values defined ends with a semi-colon.

CSS Selectors:

CSS selectors are used to "find" (or select) the HTML elements you want to style. We can divide CSS selectors into five categories:

  • Simple Selectors
  • Combinator Selectors
  • Psuedo-Class Selectors
  • Psuedo-elements Selectors
  • Attribute Selectors

CSS Class Precedence:

In this section we are going to learn about the CSS class precendence.

let us create a heading h1 with CSS class as red-text. <style>
.red-text {
color: red;
}
</style>
<h1 class="red-text">
Hello World in Red
</h1>
 Above code, will create an output like this.

Hello World in Red

Now, lets create one more header h1 but we will define two two classes in it to change the text-color to orange.

<style>
.red-text {
color: red;
}
.orange-text {
color: orange;
}
</style>
<h1 class="red-text orange-text">
Hello World in Orange. I`m preceded over Red.
</h1>

Hello World in Orange. I`m preceded over Red.

The above example illustrates clearly that class="orange-text" has taken precedence over class="red-text" since it has been declared later.

To under in a better way class="class1 class2 class3", class3 will take precedence over class2 which in turn will take precedence over class1.

we can override the class precendence by using the override keyword when defining the classes. Please see the below code for example.

<style>
.cyan-text {
color: cyan !important;
}
.blue-text {
color: blue;
}
</style>
<h2 class="cyan-text blue-text">
Hello World in blue. I`ll still be cyan.
</h2>

Hello World in blue. I`ll still be cyan.
